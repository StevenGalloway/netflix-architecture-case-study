# Runbook: Regional Evacuation

## Detection
- Regional health checks failing
- Elevated error rates
- Cloud provider incident alert

## Immediate Mitigation
1. Trigger traffic shift from impacted region
2. Freeze deployments
3. Scale healthy regions
4. Notify global incident team

## Verification
- Confirm traffic stabilization
- Monitor QoE metrics
- Validate data consistency

## Post-Incident
- Root cause analysis
- Update DR playbooks
- Schedule chaos tests
