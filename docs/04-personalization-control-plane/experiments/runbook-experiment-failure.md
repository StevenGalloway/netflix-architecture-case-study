# Runbook: Experiment Failure

## Detection
- Guardrail breach alerts
- Metric anomalies
- User complaints

## Immediate Mitigation
1. Disable experiment
2. Revert configs
3. Validate playback metrics

## Investigation
- Identify impacted cohorts
- Review experiment logic & data
- Document findings

## Recovery & Follow-up
- Implement fixes
- Schedule re-test
- Update safeguards
