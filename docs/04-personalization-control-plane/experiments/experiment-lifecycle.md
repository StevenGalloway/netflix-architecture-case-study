# Experiment Lifecycle

## 1. Hypothesis Definition
Clear problem statement and expected outcome.

## 2. Metric Selection
Primary metrics: engagement, retention, conversions  
Guardrails: TTFF, rebuffer ratio, error rate, churn proxies

## 3. Cohort Assignment
Randomized, stratified by region, device, plan.

## 4. Exposure & Execution
Feature flags control real-time exposure.

## 5. Monitoring
Continuous evaluation of metrics & guardrails.

## 6. Decision
Automatic promote / rollback / extend experiment.

## 7. Post-Experiment Review
Document results and learnings.
