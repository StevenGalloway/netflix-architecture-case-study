# Abuse Cases: Personalization Platform

## Recommendation Manipulation
Attackers attempt to bias ranking results.

**Mitigation:** Anomaly detection, model audits, canary deployments.

## Experiment Sabotage
Unauthorized experiment configuration changes.

**Mitigation:** RBAC, approvals, immutable configs.

## Data Poisoning
Injection of malicious engagement data.

**Mitigation:** Ingestion validation, outlier detection.
